## ¿Cómo utilizar este repositorio?

Puedes clonar este repositorio en tu máquina local utilizando el siguiente comando:

```
git clone https://github.com/DiegoDG-01/Alura_Inmersion-De-Datos.git
```



Una vez que hayas clonado el repositorio, puedes abrir los cuadernos Jupyter en tu entorno de desarrollo preferido para revisar los apuntes y ejecutar los ejercicios. Asegúrate de tener todas las bibliotecas de Python necesarias instaladas antes de ejecutar los cuadernos.
